<h2 align="left">🧠 Contenidos</h2>

###

<p align="left">En estos proyectos se pueden encontrar temas como:<br><br>- Programacion orientada a objetos. (PHP)<br>- Estructuras de datos y algoritmos.<br>- Base de Datos. (SQL RELACIONAL)<br>- Desarrollo Web.<br>- Patrones de diseño. (MVC).<br>- APIs. (SOAP, REST)<br>- Metodologías agiles (scrum, kanban, agile, waterfall, etc)<br>- Ciclo de vida del desarrollo de software. (SDLC)<br>- Redes y comunicaciones.<br>- Protocolos. (HTTP/S, TCP/IP, UDP, SSH, FTP)<br>- Ingles. <br>- Seguridad de los sistemas.<br>- Indicadores de rendimiento. (KPI).<br>- Cuadro de mando. (Dashboard/ CMI)<br>- Apache superset.<br>- Testing/Automatizaciones.<br>- Backend/Frontend.<br>- Figma.<br>- Trello/Jira.<br>- Postman.<br>- Agentes LLM.</p>

<h4 align="left">👱🏻‍♂️ Sobre mí</h4>

###

<p align="left">🌱 Hoy en día tengo un enfoque principal en desarrollo backend, aunque también cuento con experiencia en frontend.<br><br>Me especializo en la creación de API rest utilizando el stack LAMP (Linux, Apache, MySQL, PHP), y actualmente estoy ampliando mis habilidades desarrollando aplicaciones con MERN (MongoDB, Express, React, Node.js).<br><br>Además, estoy incursionando en: testing, agentes llm, automatizacion de procesos.<br><br>Me interesa seguir creciendo como desarrollador, aplicando buenas prácticas y participando en proyectos que representen desafíos técnicos.<br><br>💚 Posdata: actualmente estoy con proyectos como EstacionAPP, QuokkaHub y estoy empezando mi propio negocio de desarrollo web.
